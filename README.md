A project that displays daily and weekly weather forecasts for Taiwan, leveraging APIs from the Central Weather Administration (CWA) of Taiwan. Using Git Flow in a collaborative development environment, the project showcases professional development practices including feature branches, pull requests, code reviews, and conflict resolution.

## Design & Documentation

[View Figma Design](https://www.figma.com/design/jJGS8aG8uueHXgw6U3VbU8/Weather-Forecaster?node-id=215-3&t=WGwtsbcaYsOf28Wl-1)

## About The Project

This 5-day development project demonstrates a team-based approach to building a weather information platform. You can review:

- The development process through [pull requests](https://github.com/arthur3000-tw/team-project/commits/develop)
- My contributions in the [commit history](https://github.com/arthur3000-tw/team-project/graphs/contributors)

The application provides real-time and weekly weather forecasts for easy access to weather conditions across Taiwan. Users can view location-based weather information, interact with a temperature-coded map, and access detailed forecasts for each city/county.

## Key Features

- Real-time local weather information based on user location
- Interactive temperature map with color-coded regions
- Click-to-view detailed weather information for each city/county

## Technologies & Architecture

- Frontend: HTML, CSS, JavaScript
- Backend: Python FastAPI with MVC pattern
- APIs: CWB Open Data Platform, Discord Webhook, Google Geolocation
- Deployment: AWS EC2
